# Jeevan AI 🚑🌾

Jeevan AI is an AI-powered emergency healthcare platform designed for farmers, rural communities, and ASHA workers. It provides instant first-aid guidance, nearby hospital discovery, emergency support, multilingual assistance, and offline accessibility to help users during critical situations.

## Features
- AI Emergency Assistant
- Nearby Hospital Locator
- SOS Emergency Support
- ASHA Worker Dashboard
- Multilingual Assistance
- Offline Emergency Guidance

Built to bridge the rural healthcare gap and deliver life-saving support when it matters most.
